# \<my-todo>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i my-todo
```

## Usage
```html
<script type="module">
  import 'my-todo/my-todo.js';
</script>

<my-todo></my-todo>
```

## Testing using karma (if applied by author)

```bash
npm run test
```

```bash
npm run test:bs
```
